dribble.tf

In browser interactive 3D tool for reviewing past Team Fortress 2 matches. A feature lacking from the game itself, the tool allows players to analyse gameplay and provide an intuitive interface for coaching.

invUI

Exploring and recreating user interfaces from various video games. I've always loved the mix of style, usability and fun behind a good video game UI. This is a fun attempt to recreate some of these experiences in the browser.

apex pred tracker

Twitch extension that tracks the top 750 players in Apex Legends in real time. The game itself doesn't have an official leaderboard - so this is way to keep up with the game's competitive ranked scene while watching Twitch.
